# Quick Task 260622-7t4: Summary-Only HTML Export Button Summary

**One-liner:** Added `summary_only` flag to `build_html_report` and an EXPORT SUMMARY button in `app.py` that emits a compact HTML report (question + executive summary + provenance, no results table, no SQL).

## What Was Done

This was a capture-and-commit task. The feature was already fully implemented in the working tree. Execution steps:

1. Ran `python -m pytest tests/test_html_export.py -q` — 8 passed (includes the 3 new summary_only tests).
2. Ran `python -m pytest tests/ -q` — 121/121 passed.
3. Committed atomically: `src/utils.py`, `app.py`, `tests/test_html_export.py`.

## Feature Description

### src/utils.py — `build_html_report` keyword-only `summary_only` flag

- Signature: `build_html_report(..., *, summary_only: bool = False)`
- When `summary_only=True`: omits the Results table and Generated SQL sections; keeps question, executive summary, and provenance footer.
- When no summary exists and `summary_only=True`: emits an explicit "No executive summary was generated" note instead of a blank section.
- Full-report path unchanged — results/SQL refactored into locals, no behavioral difference.

### app.py — `display_results` EXPORT SUMMARY button

- Export row is `st.columns(3 if _has_summary else 2)`.
- `col[0]` = EXPORT CSV (unchanged).
- `col[1]` = EXPORT HTML (full report, unchanged).
- `col[2]` = EXPORT SUMMARY (only rendered when `executive_summary` is truthy).
- Calls `build_html_report(..., summary_only=True)`, filename prefix `incident_summary`, extension `html`, key `export_summary_{query_id}`.

### tests/test_html_export.py — 3 new tests + regression guard

| Test | Assertion |
|------|-----------|
| `test_summary_only_omits_results_and_sql_keeps_summary` | Results table + Generated SQL absent; question + summary present |
| `test_summary_only_without_summary_states_absence` | Valid doctype; explicit absence note present |
| `test_full_report_still_includes_results_and_sql` | Full report regression — results + SQL still present |
