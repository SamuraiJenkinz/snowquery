# Quick Task 260621-c28: Fix Assistant Message HTML Render Parity — Summary

**One-liner:** Add `unsafe_allow_html=True` to the history render path in `render_chat_history` so route-badge / results-count `<span>` HTML renders correctly on every Streamlit rerun, not just the first; AST-lock regression test prevents recurrence.

## What Was Done

Captured and committed a pre-existing fix to a render-parity bug in `app.py` and its companion AST-lock regression test in `tests/test_render_parity.py`.

### Root Cause

Assistant message content is app-built HTML containing route-badge and results-count `<span>` elements produced by `process_query`. This content is rendered at two sites:

| Render site | Location | Previous state | Fixed state |
|---|---|---|---|
| Live path | `render_main_content` ~line 928 | `st.markdown(response["content"], unsafe_allow_html=True)` — **correct** | Unchanged |
| History re-render | `render_chat_history` ~line 566 | `st.markdown(message["content"])` — **missing flag** | `st.markdown(message["content"], unsafe_allow_html=True)` |

On the first Streamlit render the live path fires and HTML renders correctly. On every subsequent rerun Streamlit replays from session state via the history path — which omitted `unsafe_allow_html=True` — so `<span>` tags appeared as literal markup instead of styled badges.

### Fix

Single-character-class change: added `unsafe_allow_html=True` keyword argument to the `st.markdown(message["content"])` call in `render_chat_history`.

### Regression Lock

`tests/test_render_parity.py` contains two AST-walk tests:

- **`test_content_render_sites_exist`** — asserts at least 2 `st.markdown(<name>["content"])` call sites exist in `app.py` (guards against the test becoming vacuous if render sites are renamed).
- **`test_all_content_render_sites_pass_unsafe_allow_html_true`** — walks `app.py`'s AST and fails if any `st.markdown` call with a `<name>["content"]` positional argument (where `name` is in `{message, response}`) does not also carry `unsafe_allow_html=True` as a keyword. Divergence cannot recur silently.
