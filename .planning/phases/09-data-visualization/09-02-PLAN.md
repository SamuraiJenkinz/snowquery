<title>Create src/ui/results.py with three editorial HTML renderers</title>
<description>
Create new file `src/ui/results.py` exporting three pure-Python HTML string builders.

**Module structure:**

- Module docstring describing the 3 renderers + XSS-safety contract + mono-boundary pattern (`.lp-mono` class, NOT selector expansion).
- Imports: `html` (stdlib) for `html.escape`; `pandas as pd` for `pd.isna` / `pd.api.types.is_numeric_dtype`. NO `streamlit` import.

**Constants (module scope):**

- `_NUMBER_COLS = frozenset({"number"})` — INC ID column → mono + right-align.
- `_SCORE_COLS = frozenset({"similarity_score"})` — mono + right-align.
- `_PRIORITY_COLS = frozenset({"priority"})` — italic.
- `_DATE_COLS = frozenset({"opened_at", "resolved_at", "closed_at", "updated_at"})` — italic, slightly smaller.
- `_SHORT_DESC_COLS = frozenset({"short_description"})` — min-width 320px, 140-char truncation with hover title=.
- `_TRUNCATION_CAP = 50` — DVZ-03 user-approved deviation from REQUIREMENTS.md `>1000`.
- `_SHORT_DESC_MAX_CHARS = 140` — visible cell length; full text in title= attr.
- `_PRIORITY_COLUMN_ORDER = ("number", "short_description", "priority", "opened_at", "similarity_score")` — these come first in column order when present.

**Helper functions:**

- `_td_classes(col_name: str, series: pd.Series) -> str` — returns space-separated CSS class string for a `<td>` based on column name (lowercase comparison) and dtype. Returns `"lp-mono lp-et-right"` for number/score, `"lp-et-priority"` for priority, `"lp-et-date"` for date columns, `"lp-et-desc"` for short_description, `"lp-et-right"` for any other numeric column, `""` otherwise.
- `_ordered_columns(columns: list[str]) -> list[str]` — returns `[c for c in _PRIORITY_COLUMN_ORDER if c in columns] + [c for c in columns if c not in _PRIORITY_COLUMN_ORDER]`.

**`_render_editorial_table(df: pd.DataFrame) -> str`:**

Returns HTML string for the editorial hero table.

- If `df.empty`: return `""`.
- `n_total = len(df)`; `truncated = n_total > _TRUNCATION_CAP`; `display_df = df.head(_TRUNCATION_CAP) if truncated else df`.
- Reorder columns via `_ordered_columns(list(display_df.columns))`; reindex `display_df`.
- Build `<table class="lp-editorial-table">` opening, then `<thead><tr>` with `<th>{html.escape(col.replace("_", " ").upper())}</th>` per column, close `</tr></thead>`.
- Build `<tbody>` and for each row:
  - For each column: compute `classes` via `_td_classes`; get `raw_val`; normalize `cell_text = "" if pd.isna(raw_val) else str(raw_val)`.
  - If column matches `_SHORT_DESC_COLS` AND `len(cell_text) > _SHORT_DESC_MAX_CHARS`: set `display_text = cell_text[:140] + "..."` and `title_attr = f' title="{html.escape(cell_text, quote=True)}"'`. Else `display_text = cell_text` and `title_attr = ""`.
  - Build `<td{class_attr}{title_attr}>{html.escape(display_text)}</td>` where `class_attr = f' class="{classes}"' if classes else ""`.
- Close `</tr></tbody></table>`.
- If `truncated`: append `<p class="lp-et-caption">SHOWING 50 OF {n_total:,} ROWS · EXPAND BELOW FOR FULL DATA</p>` (middot is U+00B7; comma-format N via `{n_total:,}`).
- Return joined string.

**`_render_empty_state() -> str`:**

Returns a fixed HTML string:
```
<div class="lp-et-empty"><p class="lp-et-empty-label">NO RESULTS</p><p class="lp-et-empty-body">No incidents matched your query. Try a different search or mode.</p></div>
```
Label and body text are spec-locked verbatim. Body text is charcoal (NOT muted gold) — see 09-CONTEXT.md text-contrast principle.

**`_render_chart_unavailable(feedback: str) -> str`:**

Returns:
```
<div class="lp-chart-unavailable"><p class="lp-chart-unavailable-label">CHART UNAVAILABLE</p><p class="lp-chart-unavailable-body">{html.escape(feedback)}</p></div>
```
HTML-escape the `feedback` argument defensively (it comes from `chart_generator`, but escaping is cheap insurance).

**Module exports:**

```python
__all__ = ["_render_editorial_table", "_render_empty_state", "_render_chart_unavailable"]
```

**Critical constraints:**

- NO `import streamlit`. All three functions return strings; the `st.markdown(..., unsafe_allow_html=True)` call lives at the integration site (Plan 04).
- Truncation threshold is **50**, NOT 1000. User-approved deviation; see 09-CONTEXT.md.
- Caption text spec-locked: `SHOWING 50 OF <N> ROWS · EXPAND BELOW FOR FULL DATA`. Middot is U+00B7. N comma-formatted.
- XSS: every `<td>` content goes through `html.escape()`; `title=` attrs use `html.escape(text, quote=True)`.
- NaN/None → `""` BEFORE escaping (calling `html.escape(None)` raises TypeError per Pitfall 6).
- `.lp-mono` class applied via Python on `<td>` — do NOT expand the global mono CSS selector in css.py.
